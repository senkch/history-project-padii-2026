# Генератор html странички словаря из dictionary.json

## Ключевые фишки

- Сам делает все за тебя
- Легко добавлять слова
- Поможет сдать историю

## Как пользоваться

`dictionary.json` имеет простую структуру:

- Все слова лежат в массиве `"terms"`
- Каждый элемент массива `"terms"` - это отдельное слово
- Чтобы добавить новое слово, добавь новый объект в массив `"terms"` в таком формате:

```json
{
  "term": "новое слово",
  "meaning": "значение",
  "example": "пример использования"
}
```

После изменения `dictionary.json` выполни в командной строке:

```bash
python main.py
```

Если все прошло успешно, `html` файл скомпилируется.

Приятного использования!
